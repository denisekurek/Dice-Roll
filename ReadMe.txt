Dice Roll Readme File
This will be used, not for dice, but for selecting Secret Santa recipients.

Upon starting the App / Web Page, you wile be prompted for the following:
Number or participants
How many names will be selected for each gifter
If there should be a recorded record of the gifters and giftees for future reference.

Upon completion of settings, click a button to be able to enter names for participants.
Once all names have been entered, click a button to proceed to name selection.
For giftee selection, how will the gifter be selected?
    In order of names added to the list?

For name selection, number of giftee names will be generaterated when a button is pressed on screen

There will be a logic check upon name selection to ensure the gifter did not draw their own name, and that (if multiple giftees) the giftee is not duplicated. 
    FOr example, if player A is drawing, they cannot be given themselves or multiple of Player B, C etc
